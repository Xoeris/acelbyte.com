## Project overview

The site is a static website composed of:
- HTML pages: `index.html`, `about.html`, `projects.html`, `skills.html`, `contacts.html`, and other service pages
- Assets:
  - Images and logos: `images/` (icons, titles, photographs)
  - Fonts: `fonts/`
  - Audio: `songs/`
  - Video: `videos/`
- Configuration: `CNAME` (custom domain)

## Quick preview (internal use)
- Open `index.html` directly in a web browser, or
- Run any simple static HTTP server in the project root and visit its local URL.
